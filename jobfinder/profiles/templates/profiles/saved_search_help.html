Saved Search Help Guide

What are Saved Searches?

Saved Searches allow you to save your candidate search criteria so you don't have to enter the same filters repeatedly. You can also enable notifications to be alerted when new candidates match your saved criteria.

How to Create a Saved Search

  1. Click "New Search" from the Saved Searches page
  2. Give your search a descriptive name (e.g., "Senior Python Developers in NYC")
  3. Fill in your search criteria:
    • General Keywords: Searches across all candidate profile fields
    • Location: Filter by specific location
    • Skills: Enter comma-separated skills (e.g., "Python, Django, React")
    • Project Keywords: Search for specific project types or technologies
  4. Check "Enable Notifications" if you want to be notified about new matches
  5. Click "Create Search"

Search Tips

  • You must provide at least one search criterion
  • Skills are matched with OR logic - candidates with ANY of the listed skills will match
  • Keywords search across multiple fields simultaneously
  • Location matching is flexible - "New York" will match "New York, NY"
  • Searches respect candidate privacy settings - only visible fields are searchable

Managing Your Saved Searches

Run Search

Edit Search

Delete Search

Remove a saved search you no longer need

Notifications

When notifications are enabled for a saved search, you'll receive alerts when:

  • A new candidate profile is created that matches your criteria
  • An existing candidate updates their profile to match your criteria
Note: You'll only be notified once per candidate per search. If a candidate continues to match your search, you won't receive duplicate notifications.
